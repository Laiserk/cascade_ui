# Cascade UI

Web-based UI for [Cascade](https://oxid15.github.io/cascade/en/latest/) - MLOps library for small teams and individuals.


## Installation

```bash
pip install cascade-ui
```

## How to Use

Navigate into a folder that is Cascade Workspace and run:

```bash
cascade ui
```
